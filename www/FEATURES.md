# 리튼(Litten) 기능 리스트

> 현재 frontend(Flutter) 소스 기준으로 정리한 실제 구현 기능 목록입니다. 홈페이지(랜딩 페이지) 제작의 기준 자료로 사용합니다.
> 최종 갱신: 2026-07-11

## 1. 핵심 3대 기능

| 기능 | 설명 | 상태 |
|------|------|------|
| 듣기(음성 녹음) | 녹음, 일시정지/재개, 재생, 재생속도 0.5x~2.0x, 백그라운드 녹음, 파일명 변경 | 완성 |
| STT(음성→텍스트) | 녹음 음성을 자동으로 텍스트로 변환 | 완성 |
| 쓰기(텍스트) | HTML 리치 에디터(글꼴/색상/목록/이미지) | 완성 |
| 필기(드로잉) | PDF·이미지 위에 손으로 필기, 펜 색상·굵기 조절 | 완성 |
| 첨부 파일 | 이미지·영상 등 멀티미디어 첨부 관리 | 완성 |

## 2. AI 기능

| 기능 | 설명 | 상태 |
|------|------|------|
| 자동 요약 | 텍스트·음성·PDF 자동 요약, 요약 수준(1~5) 선택 | 완성 |
| 퀴즈 생성 | 학습용 문제 자동 생성(요약 없이도 독립 생성) | 완성 |
| 유튜브 영상 요약 | 영상 자막 추출 후 자동 요약 | 완성 |
| 유튜브 채널 구독 | 채널 추가·감시(RSS), 새 영상 추적 | 완성 |
| 인사이트(리마인드) | 요약·퀴즈 기록 모아보기 | 완성 |

## 3. 생산성 도구

| 기능 | 설명 | 상태 |
|------|------|------|
| 리튼 공간 | 듣기·쓰기·필기·첨부·영상을 한 공간에 통합 관리 | 완성 |
| 캘린더·일정 | 리튼별 일정, 반복 설정, 캘린더 동기화 | 완성 |
| 알림 | 정시/10분 전/1시간 전 등 세분화 알림, 푸시·앱 뱃지 | 완성 |
| 검색·즐겨찾기 | 파일명·내용 검색, 검색 이력, 북마크 | 완성 |
| 스터디룸 | 1:1·그룹·나만의 룸에서 학습자료 공유, 룸별 [대화]·[자료] 관리 | 완성 |
| 스터디 대화 | 스터디룸 멤버와 메시지를 주고받으며 함께 학습(비실시간) | 완성 |
| PDF 변환·뷰어 | 이미지·문서를 PDF로 변환 후 필기, PDF 조회 | 완성 |

## 4. 글로벌 · 테마

| 기능 | 설명 | 상태 |
|------|------|------|
| 30개 언어 | 전 세계 30개 언어 UI 지원 | 완성 |
| 시스템 언어 자동 감지 | 설치 시 기기 언어로 자동 설정 | 완성 |
| 5가지 테마 | Classic Blue, Dark Mode, Nature Green, Sunset Orange, Monochrome Grey | 완성 |
| 지역별 테마 자동 선택 | 언어/국가에 맞는 테마 자동 적용 | 완성 |

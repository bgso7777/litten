📝 리튼의 강력한 노트 기능

듣기, 쓰기, 필기 - 세 가지 방식으로 당신의 아이디어를 기록하세요

🎵 고품질 음성 녹음

강의, 회의, 인터뷰 등 중요한 순간을 놓치지 마세요. 리튼의 듣기 기능으로 모든 대화를 선명하게 기록합니다.

주요 기능

  • 고품질 오디오 녹음 (44.1kHz)
  • 백그라운드 녹음 지원
  • 녹음 중 일시정지/재개
  • 재생 속도 조절 (0.5x ~ 2.0x)
  • 특정 구간 반복 재생
  • 오디오 파일 공유 및 내보내기

💡 Pro Tip

녹음과 동시에 텍스트나 필기 노트를 작성하여 나중에 쉽게 찾아볼 수 있습니다.

🎤 음성 녹음 화면

활용 사례

🎓

학생

강의를 녹음하고 나중에 복습할 때 중요한 부분만 골라 들으세요. 시험 기간에 큰 도움이 됩니다.

💼

비즈니스

회의 내용을 녹음하여 중요한 결정사항을 놓치지 마세요. 회의록 작성도 훨씬 쉬워집니다.

🎙️

인터뷰

인터뷰 내용을 정확하게 기록하고, 나중에 다시 들으며 중요한 인사이트를 발견하세요.

✍️ 풍부한 텍스트 편집

아이디어를 자유롭게 정리하고 표현하세요. HTML 에디터를 사용하여 블로그처럼 아름다운 노트를 작성할 수 있습니다.

주요 기능

  • 다양한 글꼴 및 스타일 적용
  • 볼드, 이탤릭, 밑줄 등 포맷팅
  • 글머리 기호 및 번호 매기기
  • 색상 및 하이라이트
  • 링크 삽입
  • 이미지 첨부
  • 표(테이블) 삽입
  • 코드 블록 지원

✨ 특별한 기능

작성한 텍스트를 PDF로 내보낼 수 있어 다른 사람과 공유하거나 보관하기 편리합니다.

활용 사례

📚

학습 노트

강의 내용을 정리하고, 중요한 개념을 하이라이트하여 효과적으로 복습하세요.

💡

아이디어 정리

떠오르는 아이디어를 빠르게 기록하고, 나중에 체계적으로 정리하세요.

📋

회의록

회의 내용을 구조화하여 작성하고, 액션 아이템을 명확하게 정리하세요.

✏️

블로그 초안

블로그 글을 작성하기 전에 리튼에서 초안을 작성하고 다듬으세요.

🎨 자유로운 필기와 주석

PDF 문서나 이미지 위에 직접 필기하세요. 손글씨로 메모하고, 중요한 부분에 주석을 달아 생각을 시각화합니다.

주요 기능

  • PDF 파일을 PNG로 변환하여 필기
  • 이미지 위에 자유로운 그리기
  • 다양한 펜 색상 및 두께
  • 형광펜 도구
  • 지우개 도구
  • 실행 취소/다시 실행
  • 확대/축소 및 팬
  • 필기 레이어 관리

🎯 강력한 기능

여러 페이지 PDF를 PNG로 변환하여 각 페이지마다 필기할 수 있습니다. 완성된 필기는 다시 PDF로 내보낼 수 있습니다.

✏️ 필기 화면

PDF → PNG 변환 후 필기

PDF 파일을 가져오면 자동으로 각 페이지를 고품질 PNG 이미지로 변환합니다.

변환된 이미지 위에서 자유롭게 펜으로 그리거나 형광펜으로 하이라이트할 수 있습니다.

멀티터치를 지원하여 확대/축소하며 세밀한 필기가 가능합니다.

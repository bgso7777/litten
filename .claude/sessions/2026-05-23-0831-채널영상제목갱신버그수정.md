# 채널 영상 제목 갱신 버그 수정

## 세션 개요
- **시작 시간**: 2026-05-23 08:31 (KST)
- **세션 유형**: 버그 수정 및 안정화

## 목표
영상 구독에서 추가된 채널들의 제목과 내용들이 갱신되지 않는 버그 수정

### 문제 분석
- `YoutubeService.java:255` 에서 `existsByVideoId(videoId)` 가 true이면 제목 변경 없이 완전히 스킵
- RSS 피드에서 기존 영상의 제목이 변경되어도 DB에 반영되지 않음
- 서버 로그: 모든 채널에서 `신규처리: 0, 스킵: 15` — 기존 영상 전부 스킵

### 수정 계획
- `existsByVideoId` → `findByVideoId`로 변경하여 기존 영상 엔티티 조회
- 기존 영상이면 제목이 변경된 경우에만 DB 업데이트 후 스킵
- 신규 영상이면 기존 로직(`processNewVideo`) 실행

### 미래 개발자를 위한 팁

1. **텍스트 에디터 초기화**: `_editorInitialized` 플래그를 확인하고 `_pendingContent`에 내용을 저장하는 패턴 유지. `onInit` 이후 200ms 딜레이 필수.
2. **STT 재시작 로직**: `_isManualStop = false`이고 `error_speech_timeout` / `error_no_match` / `error_client` 에러면 3초 후 자동 재시작. `onStatus('done')`도 동일하게 2초 후 재시작.
3. **요약 API 호출**: `ApiService.summarizeText()` — `_useMock` 플래그로 개발/운영 전환 가능.
4. **Maven 빌드**: `JAVA_HOME`을 Android Studio JBR로 설정하고 `/c/tools/maven/apache-maven-3.9.6/bin/mvn` 절대 경로 사용.
5. **API 키 관리**: `application.yml`에 절대 직접 커밋하지 말 것. 환경변수 `${CLAUDE_API_KEY:}` 사용.

**구현된 해결책**:
1. **JavaScript Interop 문제 수정**: `js.allowInterop`와 `apply` 메서드 오류 해결

   - 1차: 직접 PDF.js (async/await 방식)
   - 2차: pdf_render 패키지 (기존 방식)
   - 3차: 향상된 JavaScript (폴링 방식)
3. **안전한 JavaScript 실행**: eval 기반 async/await 코드 사용
4. **폴링 기반 결과 처리**: `window._pdfRenderComplete` 플래그 활용

**변경된 코드 내용**:
- `_renderPdfWithDirectPdfJs()`: 새로운 PDF.js 직접 호출 메서드
- `_renderPdfViaDirectJs()`: 폴링 기반 향상된 JavaScript 실행
- `_renderActualPdfContent()`: 렌더링 우선순위 변경 (PDF.js 최우선)
- 상세한 디버깅 로그 추가: 각 단계별 성공/실패 추적

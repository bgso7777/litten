#### 핵심 발견 (근본 원인)
요약 UI/플래그/시트는 이미 모두 구현돼 있었으나 한 곳에서 끊겨 3가지가 모두 미동작:
- 프론트 `_summarize()`([youtube_video_player_sheet.dart] _ScriptSummarySheetState)가
  `processSummary(fileType: 'text', youtubeVideoId: ...)` 로 호출.
- 백엔드 `SummaryProcessService.findOrCreate()/findCached()` 는
  `fileType == 'youtube' && youtubeVideoId != null` 일 때만 `youtube_video_id` 기준 저장/조회.
- 따라서 'text'로 보내면 `youtube_video_id`가 NULL로 저장 →
  `getYoutubeSummaryCache(videoId)` 조회 실패 → 아이콘 비활성 + 요약 클릭 시 내용 없음.

#### 추가 발견 (프롬프트 미설정)
- fileType을 'youtube'로 바꾸자 에뮬레이터에서 "시스템 프롬프트가 설정되지 않았습니다" 실패.
- 원인: 백엔드 `note_prompt_config` 테이블에 `fileType='text'` 행만 있고 `'youtube'` 행이 없음.
  → `defaultConfig`(prompt=null) → `OpenAiSummaryService.summarize()` L65~67 에서 systemPrompt 비어 실패.
- 실제 아이폰은 직전 설치본('text' 요청)이라 text 프롬프트로 성공했던 것(단, youtube_video_id는 NULL).

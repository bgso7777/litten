## 목표 (Goals)

### 1. 요약 기능 개선
- STT 전사 영역에 30초 단위 녹음 시간 타임스탬프 삽입
- 자동 요약 시점마다 전사 내용 바로 아래에 요약 삽입
- 최종 결과: "전사내용 + [0:30] + 전사내용 + [1:00] + 요약1 + 전사내용 + [1:30] + 전사내용 + [2:00] + 요약2..."
- 요약 영역(2/3): 최신 요약만 표시
- 과거 요약 이력: TextFile.summaryHistory 배열에 저장
- 타임스탬프 클릭 시 해당 위치부터 오디오 재생 가능하도록 메타데이터 포함

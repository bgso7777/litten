### 발생 문제와 해결
- **디스크 풀(99%, 여유 282MB)**: 앱 설치 실패 → DerivedData(2.8GB)+iOS DeviceSupport(27GB)+시뮬레이터/AVD 정리로 해결.
- **iOS 시뮬레이터 멈춤(저CPU 정지)**: `sample`로 메인스레드가 `clipboardHasStrings`→UIPasteboard XPC 동기 대기 데드락임을 규명. 원인=거대 자막(2.5만자) 클립보드 복사 + 시뮬레이터↔맥 클립보드 동기화. 해결=호스트 클립보드 비우기 + 시뮬레이터 재부팅. **실기기는 영향 없음**.
- **거대 자막 SelectableText 멈춤**: 선택영역 계산이 메인스레드 점유 → 스크립트 패널을 Text로 교체.
- **텍스트 요약 "시스템 프롬프트 미설정" 에러**: 구 엔드포인트 `/summary`가 백엔드에서 프롬프트 미구성 → `processSummary(fileType:'text')`로 위임.
- **Android 무료 카운트 리셋 불가(SELinux)**: run-as 쓰기 거부 → main에 1회 리셋 코드 임시 삽입 후 원복.
- **요약 다이얼로그 버튼 세로 줄바꿈**: 라벨 길어져 OverflowBar 줄바꿈 → 가로 Row 고정 + 라벨 단축.

### 얻은 교훈 / 미래 개발자 팁
- 시뮬레이터에서 **대용량 클립보드 복사는 pasteboard 데몬을 멈출 수 있음**. 재현 시 호스트 클립보드 비우고 시뮬레이터 재부팅. 시뮬레이터 메뉴 *Edit → Automatically Sync Pasteboard* 해제로 예방 가능.

- Android 앱 private 파일은 **SELinux로 외부(run-as) 쓰기 불가** → prefs 조작은 앱 내부 코드로.
- 무료 판별 기준은 토큰 유무가 아니라 **`subscriptionType == SubscriptionType.free`** 로 통일됨.

### 발생 문제와 해결
- **프로덕션 다운(약 13분)**: macOS tar가 `._`(AppleDouble) 파일을 서버로 올려 Spring 컴포넌트 스캔 ASM 파싱 실패 → 컨텍스트 초기화 실패. 해결: `._*` 삭제 후 재기동, 이후 배포는 **단일 .class 직접 scp + COPYFILE_DISABLE=1**.
- **Tomcat 재기동 미스**: kill 직후 같은 세션 startup이 포트 미해제로 종종 실패 → **포트 해제 대기 후 startup 분리 실행**으로 정착.
- **iOS PathNotFoundException**: 컨테이너 UUID 변경으로 절대경로 무효 → littens/ 이후 경로를 현재 Documents 기준 재구성.
- **받은 그룹 인원 2/3 불일치**: 받은 메시지에 멤버 수 없음 → 백엔드 groupMemberCount 추가.
- **호스트키 변경**: 서버 SSH ECDSA 키 변경 확인 후 accept-new로 접속(사용자 확인).

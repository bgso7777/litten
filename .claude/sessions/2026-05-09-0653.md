### 구현된 기능 상세

#### 백엔드 (Java Spring Boot)

**OpenAiSummaryService.java** — 완전 재작성
- `buildSystemPrompt(textLanguage, summaryLanguage, level)` — 규칙 전체를 system 메시지로
- `buildUserContent(text, ...)` — 콘텐츠만 user 메시지로 분리
- `computeMaxTokens(text, level)` — 텍스트 길이 기반 동적 토큰 계산
  - Lv1: 15%, Lv2: 30%, Lv3: 55%, Lv4: 80%, Lv5: 100% + 800 오버헤드
- 리마인드 3단 계층 출력 규칙 포함 (`📂 → ▸ → └` 형식)
- 수준별 섹션 및 문장 수 명시

**SummaryService.java** — Claude API 버전도 동일하게 개선
- `computeMaxTokens(text, level)` 추가
- `application.yml` max-tokens 1024 → 8192

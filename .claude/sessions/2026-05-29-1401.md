4. **유튜브 스크립트/요약 흐름**: 데스크톱 유튜브 watch를 WebView로 띄워 네이티브 "스크립트 표시" 자동 오픈 → "스크립트 복사 → 요약하기"로 추출/복사 후 바텀시트(채널·제목·일자 헤더 + 스크립트 + 요약설정 + summarizeText 비로그인 요약).

### 발생한 문제와 해결
- 클라이언트 자막 직접추출=BotGuard 차단(과거 05-23~25 세션 교훈 재확인) → 유튜브 네이티브 UI에 맡기고 사용자 복사 방식 채택(저작권+BotGuard 회피).
- 모바일 웹엔 "스크립트 표시" 없음 → 데스크톱 UA + www.youtube.com/watch.
- iOS WebView 영상 자동 전체화면 → allowsInlineMediaPlayback + JS playsinline 강제.
- 자동 스크립트 버튼 클릭이 숨김/중복요소 눌러 무동작 → visible 요소만, is-expanded 토글방지, LittenDiag 진단채널.
- 자막 세그먼트 셀렉터 부재 → 트랜스크립트 패널 innerText 추출 + 타임스탬프/헤더 정리.
- 팝업 스크롤이 뒤 WebView로 누수 → WebView 투명 오버레이로 touchmove/wheel 차단.
- 전체탭 영상섹션 무한 리로드 → `_youtubeAutoLoadRequested` 1회 로드 가드.

### 얻은 교훈 / 미래 개발자 팁
- **유튜브 자막은 클라이언트 스크래핑으로 풀지 말 것** — BotGuard/PoToken으로 막힘. 안정 경로는 백엔드(yt-dlp/Supadata, 로그인 필요) 또는 "네이티브 UI + 사용자 복사".
- 생성자 시그니처 변경은 핫 리로드로 반영 안 됨 → **핫 리스타트(USR2)** 필요(`NoSuchMethodError: No constructor` 발생 시).
- iOS WebView 위 Flutter 버튼은 **WebView와 겹치면 탭이 새므로 별도 바로 분리**, 모달 위 스크롤 누수는 WebView 내 오버레이로 차단.
- `summarizeText`는 토큰 불필요 → 비로그인 요약 가능. 자막/요약 저장(`saveYoutubeTranscript`)만 토큰 필요.
- 미사용 잔존 코드: `youtube_video_player_sheet.dart`의 `_onSummaryTap`/`_CopyrightConsentDialog`/`_YoutubeSummaryDialog`, `youtube_http_transcript_service.dart`의 timed 메서드(`fetchTimedSegments` 등) — 정리 대상.
